# gooser
A JSON transformer written in go
## Running Locally
- git clone https://github.com/djfunksalot/gooser.git
- cd gooser
- go build -o main .
- ./main
## Running on replit
[https://replit.com/@djfunksalot/gooser](https://replit.com/@djfunksalot/gooser)
click on "Run" button at the top of the page
